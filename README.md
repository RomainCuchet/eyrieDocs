
# eyrieDocs

Website: https://romaincuchet.github.io/eyrieDocs/

eyrieDocs' mission is to publish cybersecurity reports and tools, empowering you to hack ethically to build a more resilient and safer digital world for everyone. All entities are contacted and reports sanitized. When responses fail, we advocate for user safety through transparent awareness.

Access sanitized disclosures and tools in the Docs or explore the Blog for penetration test write-ups and security findings.
